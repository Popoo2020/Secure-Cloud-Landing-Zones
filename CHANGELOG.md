# Changelog

This file records significant changes to **Secure‑Cloud‑Landing‑Zones** using
the [Keep a Changelog](https://keepachangelog.com/) format.

## [0.1.0] – 2026‑03‑01

### Added

* **Azure identity baseline documentation:** Added
  `docs/azure_identity_baseline.md` describing identity management best
  practices, privileged identity management (PIM) concepts and conditional
  access patterns for Azure environments.
* **Control objectives mapping:** Added `docs/control_objectives.md` to
  describe how cloud landing zone components align with high‑level control
  objectives (e.g. identity, network, logging and key management).
* **Repository standards:** Added license, security policy, code of
  conduct, contributing guidelines and code owner definitions.
* **Initial release:** Tagged version `v0.1.0` as the first structured
  release.
